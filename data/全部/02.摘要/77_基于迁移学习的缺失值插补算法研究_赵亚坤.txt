摘要
随着大数据时代的到来，数据分析逐渐显露出其巨大的价值。作 为数据分析的重要组成部分，缺失值插补技术直接影响数据分析的准 确率，进而影响数据分析的结果。关于缺失值插补的相关工作，已经 有很多，然而，关于缺失值数量较多场景下的缺失值插补算法的研究 目前还十分缺乏。而实际工业生产中，往往产生数量较多，比例较大 的缺失值。因此，提高缺失数量较多的缺失值插补算法准确率成为了 当下缺失值插补算法的热点问题。本文针对这个问题，做了以下工作。
首先，针对网格类数据场景下某一特征缺失值数量过多的情况， 分析了迁移学习算法TradaBoost对其进行缺失值插补的可行性，并 与k-NN,低秩矩阵插补和0值插补等传统缺失值算法进行性能对比 分析。在糖尿病预测数据集上的测试结果表明，在AUC和F1指标上， 基于TradaBoost的缺失值插补算法相比于传统算法取得的效果不明 显。这是因为TradaBoost只能接受离散变量输入，用之构建缺失值算 法需要将连续变量转化成离散变量，丢失大量信息。
接下来,针对TradaBoost缺失值插补算法性能提升不理想的问题， 进一步提出了基于回归的TradaBoostReg算法，减少了训练过程中连 续值转化成离散值导致的信息损失。在糖尿病预测数据集上的实验仿 真表明，在缺失值比例大于51 %的时候，TradaBoostReg算法相较于 传统缺失值插补算法性能更优。
最后，针对算法迭代效率低的问题，对回归TradaBoostReg算法的 损失函数进行标准化处理，使得误差损失一直工作在最佳范围内。在 糖尿病预测数据集上的实验仿真表明，在保证性能较于传统缺失值算 法略有提升的情况下，大大提高了迭代的效率。
关键词：缺失值插补迁移学习回归误差标准化
RESEARCH ON MISSING VALUE INTERPOLATION
ALGORITHM BASED ON TRANSFER LEARNING
ABSTRACT
With the advent of the era of big data, data analysis has gradually revealed its great value. As an important part of data analysis, missing value interpolation technology directly affects the accuracy of data analysis and the results of data analysis. There have been many related works on missing value interpolation. However, there is still a lack of research on missing value interpolation algorithms in scenarios with a large number of missing values. In actual industrial production, there are often a large number and a large proportion of missing values. Therefore, improving the accuracy of the missing value interpolation algorithm with a large number of missing data has become a hot issue of the current missing value interpolation algorithm. In response to this problem, this article has done the following work.
First,for the situation where the number of missing values of a certain feature in the grid-like data scenario is too large, the feasibility of the transfer learning algorithm TradaBoost to interpolate the missing values is analyzed. Performance comparison and analysis of traditional missing value algorithms such as value interpolation. The test results on the
diabetes prediction data set show that on the AUC and Fl indicators, the missing value interpolation algorithm based on TradaBoost is less effective than the traditional algorithm. This is because TradaBoost can only accept discrete variable input, and the algorithm used to construct the missing value needs to convert continuous variables into discrete variables and lose a lot of information.
Next, in view of the problem of TradaBoost missing value interpolation algorithm performance improvement is not ideal, a TradaBoostReg algorithm based on regression is further proposed to reduce the information loss caused by the conversion of continuous values into discrete values during training. Experimental simulations on the diabetes prediction data set show that when the proportion of missing values is greater than 51%, the TradaBoostReg algorithm performs better than the traditional missing value interpolation algorithm.
Finally, for the problem of low iteration efficiency of the algorithm, the loss function of the regression TradaBoostReg algorithm is standardized to make the error loss always work in the optimal range. Experimental simulations on the diabetes prediction data set show that, under the condition that the performance is slightly improved compared with the traditional missing value algorithm, the efficiency of iteration is greatly improved.
KEY WORDS: Missing value interpolation Transfer learning Regression Eiror standardization
